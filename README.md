# ozzx-example

下载项目文件
```

安装依赖：
```
cd example
npm i 或 yarn
```
安装ozzx：
```
npm i -g ozzx
```
运行:
```
pack dev
```
编译:

pack build

在页面中使用模板:
```
<template>
  <temple name="topBar" src="http://cunchu.site/github/ozzx/topBar.page"></temple>
  <div class="example">样式隔离:组件与组件之间的样式不会相互影响</div>
</template>
```
